# Shop Bot
This is a Telegram e-commerce bot built with aiogram 3.x.

## Features
- Product catalog
- Cart management
- Checkout process
- Order handling using inline keyboards

## Setup
1. Clone the repository.
2. Install the requirements using `pip install -r requirements.txt`.
3. Create a `.env` file based on `.env.example`.

## Usage
Run the bot with:
```bash
python main.py
```
